# jpashop 
### JPA를 이용한 간단한 쇼핑몰입니다.

<br></br>
### **기능**
- 회원 기능
   - 회원 등록
   - 회원 조회
- 상품 기능
  - 상품 등록
  - 상품 수정
  - 상품 조회
- 주문 기능
   - 상품 주문
   - 주문 내역 조회
   - 주문 취소
